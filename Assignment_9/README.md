# Assignment 9 - Spring Boot Hello World

This project is a simple Spring Boot web application that displays a `Hello World` message on a webpage.

## Features

- Embedded Tomcat server through Spring Boot
- Browser-accessible Hello World webpage at `/`
- Simple REST endpoint at `/api/hello`

## Run the project

```bash
mvn spring-boot:run
```

Then open:

- `http://localhost:8080/`
- `http://localhost:8080/api/hello`
